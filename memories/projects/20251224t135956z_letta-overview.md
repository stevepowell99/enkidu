Concise canonical summary: Letta is a platform for building stateful AI agents with long-term memory and advanced reasoning. Key product lines are the Developer Platform (APIs), Letta Code, and a Learning SDK / ADE for building agent behaviours. The platform emphasises long-term memory for agents, developer APIs, and tooling for packaging and running agent logic.
