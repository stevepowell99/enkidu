- The app uses a two-pane layout with a draggable border between them (default split 30:70).
- Left-hand side is for selecting sources and creating/filtering links.
- Right-hand side (pink tabs) presents the results.
- Left side features: Project Dropdown Menu, Sources Dropdown Menu, Create Links tab, Filter Links tab.
- The Links Pipeline maps the left-side steps to the results displayed on the right.
- Right-hand outputs include Map, Factors, Links, Pivot Tables.
- Right-hand side tabs include Help, Projects, Sources, Settings, Account, Logs, Bookmarks, Responses.

### Key quotes
- "The left hand side of the app is all about selecting sources then creating and filtering links."
- "The right hand side (the pink tabs) is all about presenting the results."
