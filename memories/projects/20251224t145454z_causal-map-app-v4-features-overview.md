- CM4 improves speed, stability, and offline resilience; scalable to hundreds or thousands of concurrent users.
- New data model: each source text is treated as a single text; there is no separate "statement" concept in cm4.
- Custom Columns support with a spreadsheet-like editor; easier import, tagging, and complete project management; PDF/DOCX uploads supported.
- Bulk editing for labels in the links and factors tables.
- Enhanced filtering and analysis: multi-version filters, optional semantic filters like cluster and soft recode (requires AI subscription); pivot tables and graphs for data analysis.
- Sharing and collaboration: anonymous login for viewing without login; URL-based state saving; Team-subscription gated features include real-time collaboration and interactive maps.
- AI Coding: optional AI-powered coding assistant; "Human first, AI next".
- CM3 features not implemented: deep cross-source question support; viewing multiple sources simultaneously.
