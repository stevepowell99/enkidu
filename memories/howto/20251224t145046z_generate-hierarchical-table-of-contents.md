- Generate a hierarchical table of contents for {} using appropriate heading levels (H1, H2, H3, etc.).
- Include page numbers if present.
- Return only the table of contents; do not output extraneous text.
- Apply default Copilot front-matter toggles (context-menu, slash-enabled, order) as guidance when generating the TOC.
