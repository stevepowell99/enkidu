- Convert the instruction into a single tweet under 280 characters.
- Use concise, impactful language that preserves the core message.
- Drop meta-prompt details and extraneous context.
- Ensure the tweet is self-contained and understandable without extra context.
- Do not include anything beyond the tweet text in the output.
