- The Factors panel lists all unique labels from the current filter pipeline, with metrics like Citation Count, Source Count, and Citation Count: In/Out, Source Count: In/Out, sorted by total citations (desc).
- Table features include columns for the metrics, click-to-select (no checkbox), server-side pagination, and an actions column with an edit button to open the factor edit modal.
- Action Buttons: Delete, Relabel, Search/Replace, Merge; buttons are disabled until factors are selected. Tip: use Search/Replace for quick, scoped relabeling.
- Bulk factor labels editor lets you bulk edit labels: toggle Bulk Edit, multi-cursor editing, line-by-line edits, recoded labels are read-only, and unsaved edits are discarded when you sort or filter. Press Save Changes to apply after a confirmation showing affected links.
- Demographic Breakdowns and Statistical Significance Testing: analyze by demographics with breakdowns, and perform chi-squared tests to identify group differences; options and visual indicators described.
