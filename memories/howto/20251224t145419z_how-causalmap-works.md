- Create Projects: organize sources (PDFs/docs) into projects; each project can hold multiple sources.
- Create Causal Links manually: highlight causal claims to create links, then view them as a causal map.
- Create Causal Links using AI: leverage AI Coding to generate links.
- Filter & Analyze: apply Analysis Filters to focus on data; explore Map, Factors, Links, and Statistics.
- Collaborate: work with multiple users and manage sharing in Sharing and Permissions.
