- Quick reference for selecting and managing projects in the Causal Map app: project selector header, dropdown, and the Project Details screen.\n- File menu actions for the current project: Edit, Upload sources, Clone, Archive, Delete all links, Download, Versions, and Manage projects.\n- Link-sharing aids: Copy link (bookmark URL) and Copy formatted link (HTML) to share current app state.\n- Visibility and access: archived projects are read-only; admins can see all but non-viewable projects won't auto-load; Locked/Public/Collaborators update permissions instantly.\n- Project Details capabilities: editing Name/Tags/Description, codebook, AI Processing Region, Show on open, Sharing (Locked/Public/Collaborators), and Bookmarks; AI region defaults to EU and persists per project.\nSource: content/999 Causal Map App/050 Projects Bar ((project-selector-header)).md
