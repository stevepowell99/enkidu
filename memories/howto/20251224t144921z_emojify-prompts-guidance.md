- Purpose: Provide rules to add relevant emojis to enhance text content without changing its meaning
- Rules: 1) Insert emojis at natural breaks in the text; 2) Never place two emojis next to each other; 3) Keep original text unchanged; 4) Choose emojis that match context and tone
- Output constraint: Return only the emojified text
- Usage note: Use emojis sparingly to avoid clutter
