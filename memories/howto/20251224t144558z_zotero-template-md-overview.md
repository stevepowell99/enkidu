- YAML front matter fields like cssclass, type, title, publication, date, archive, archive-location, and citekey.
- Creators are grouped by creatorType and mapped to author; names are formatted as Last, First when needed.
- Bibliography, abstractNote, and online/local URLs are rendered; PDFs are linked if attachments end with .pdf.
- Tag-based path mapping writes notes into source/secondary, source/primary, project, or subject folders depending on tag values.
- Note section defines a color-coded calloutHeader macro used for Important, Reference, and Undefined labels.
- A persist annotations block collects Zotero annotations and displays them with optional images, annotated text, and page references.
